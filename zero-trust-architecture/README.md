# Advanced Network Security: Zero Trust Architecture

## Objective
To implement Zero Trust principles in a local network by segmenting the network and enforcing strict access controls.

## What is Zero Trust?
Zero Trust is a cybersecurity model based on “never trust, always verify.” Every user, device, and request must be authenticated before access is granted.

## Tools Used
- Windows Firewall
- Local network configuration
- Access control policies

## Network Segmentation
The network was divided into three zones:

1. User Zone – Regular users with limited access  
2. Admin Zone – Administrator-level access  
3. Restricted Zone – Sensitive resources  

## Access Control
- Only authorized users can access resources  
- Least privilege principle applied  
- Admin access restricted  
- Unnecessary permissions removed  

## Firewall Rules
- Block all unnecessary inbound traffic  
- Allow only required outbound traffic  
- Restrict access to sensitive services  

## Zero Trust Principles Applied
- Never trust, always verify  
- Least privilege access  
- Network segmentation  
- Continuous monitoring

## Conclusion
Zero Trust architecture improves security by limiting access, verifying users, and reducing attack surfaces.
